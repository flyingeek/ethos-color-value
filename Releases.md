# 1.0.0

This is the first release for the Color Value Widget for Frsky Ethos.

## Download and Install

You can download the file in the assets below.

You can install the zip file using the Ethos suite App, or you can unzip and drag the folder named _color-value_ in your radio's script folder. After installation two new widgets are be available: _Color Value_ and _Color Telemetry_, both share the same code but _Color Telemetry_ restrict the Source of the value to be a telemetry sensor.
